# CNC Machine Prototype

This project involved designing, manufacturing, and assembling a functional CNC machine prototype.

The machine was designed in **SolidWorks**, where each component was modeled to ensure proper fit and assembly. The parts were then manufactured using **3D printing** and assembled into a working prototype, providing hands-on experience in the complete prototyping process.

## Project Highlights

* Designed the CNC machine and individual components using SolidWorks,
* Prepared parts for 3D printing,
* Manufactured components,
* Assembled the printed parts into a functional CNC machine prototype,
* Verified the mechanical assembly and overall functionality.

## Skills Demonstrated

* CAD Design (SolidWorks),
* Mechanical Design,
* 3D Printing,
* Rapid Prototyping,
* Mechanical Assembly,
* Engineering Design.

## What I Learned

This project strengthened my understanding of the engineering design process by taking a concept from CAD modeling to a physical, functional prototype. It also improved my practical skills in designing for manufacturability, component assembly, and translating digital models into working hardware.

This project reflects my interest in mechanical design, automation, and robotics, while demonstrating the integration of CAD, additive manufacturing, and prototype development.
